# Stellar - Web3 Wallet Portal

Stellar is a modern Web3 portal that enables users to securely connect their MetaMask wallet and interact with decentralized blockchain networks. The platform provides a simple and intuitive interface for wallet authentication, balance monitoring, and network information management.

## 🚀 Features

- MetaMask wallet integration
- Secure cryptographic signature authentication
- Real-time wallet balance display
- Wallet address visualization
- Connected network information
- Easy wallet connection and disconnection
- Responsive and user-friendly interface
- Beginner-friendly Web3 experience

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript 
- Web3.js / Ethereum APIs
- MetaMask Wallet Integration

## 📋 Functionalities

### Wallet Connection
Users can securely connect their MetaMask wallet through the browser extension.

### Authentication
The application uses cryptographic signature-based authentication to verify wallet ownership.

### Balance Tracking
Displays the current wallet balance fetched directly from the blockchain network.

### Network Information
Shows the currently connected blockchain network and wallet address.

### Wallet Management
Allows users to easily connect or disconnect their wallet session.

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/rajsen05/stellar-web3-portal.git
```

2. Navigate to the project directory:

```bash
cd stellar-web3-portal
```

3. Open the project in your preferred code editor.

4. Launch the application:

```bash
open index.html
```

or use a local development server.
